# Academic PDF Translator v0.1.0（macOS Apple Silicon）

一款面向学术阅读的 PDF 翻译工具。  
核心体验是“边读边译”：打开 PDF 后，把鼠标悬停在英文词语上即可看到中文翻译，尽量不打断阅读流程。

## 本版功能
1. 本地 PDF 打开与阅读
2. 英文词语悬停翻译（本地模型 + 词典兜底）
3. 触控板左右滑动翻页
4. 鼠标侧键翻页（常见 3/4 键）
5. 翻页按钮悬浮显示，支持位置切换（右下/左下/右上）
6. 缩放改为 `+ / -` 按钮，并实时显示缩放百分比
7. 中文默认界面，去除冗余整页/全文翻译面板，聚焦阅读主流程

## 安装与启动（macOS）
1. 下载并解压 zip，得到 `Academic PDF Translator.app`
2. 拖到 `Applications` 文件夹
3. 首次打开若被系统拦截：
   - 右键 App -> `打开` -> 再次确认 `打开`
   - 或前往 `系统设置 -> 隐私与安全性` 点击“仍要打开”

## 使用建议
1. 打开论文 PDF 后，直接把鼠标移动到英文词上看翻译
2. 扫描版/图片型 PDF 可能文本层不完整，悬停翻译效果会受限
3. 首次模型加载可能稍慢，后续会使用缓存

## 已知限制
1. 当前发布仅支持 Apple Silicon（M1/M2/M3）
2. 应用暂未签名与公证，首次打开可能出现系统安全提示
3. 对“纯图片 PDF”的识别能力有限（后续可加入 OCR 模式）
